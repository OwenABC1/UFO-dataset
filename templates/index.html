About The Data

The data used on this website is from the CORGIS Dataset Project.
The data contains statistics about UFO sightings from 1910 to 2014.

The Dataset includes statistics on the duration and location of each encounter, as well as a very brief description of the UFO itself.
For example a ufo encounter could have the following data:

-Country: US
-State: AK
-City: Anchor Point
-Duration: 300 Seconds
-UFO shape: Disk
-Description: Large UFO over Mt. ILIMANA Alaska.
